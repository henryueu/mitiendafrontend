###  Tienda Don José 
**Descripción:** Sistema Full Stack para la gestión de ventas e inventarios en micronegocios. Desarrollado con Node.js, Express y PostgreSQL (Neon), permite el control transaccional de productos y seguridad basada en roles para asegurar la integridad operativa.
